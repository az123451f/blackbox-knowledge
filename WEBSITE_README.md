# The Chosen Frequency Discord Server Website

## Overview
This is a professional website for "The Chosen Frequency" Discord server - a high-standard self-development community focused on mindset, physical strength, and spiritual growth.

## Features Included
- **Modern, Responsive Design**: Works on all device sizes
- **Complete Sections**:
  - Hero section with call-to-action
  - About section
  - Features section highlighting all community aspects
  - Latest announcements section
  - Join section with prominent CTA
- **Professional Styling**: Using CSS Grid, Flexbox, and modern UI principles
- **Performance Optimized**: Lightweight and fast-loading

## Files Included
- `index.html`: Main HTML structure with all content
- `styles.css`: Complete styling for the website
- `assets/`: Folder for future images and resources

## How to Run the Website

### Option 1: Using Python HTTP Server (Recommended)
1. Open terminal in the `/workspace` directory
2. Run: `python3 -m http.server 8000`
3. Visit: `http://localhost:8000` in your browser

### Option 2: Local Development
- Simply open `index.html` in any web browser
- Or use your preferred development server (VS Code Live Server, etc.)
